# 桌面清理小助手 · 使用说明

一个 Windows 小工具：**自动整理桌面图标** + **强力卸载顽固软件**。

## 一、第一次怎么跑起来（开发/源码方式）

1. 确认装了 Python 3.10 以上（命令行输入 `py --version` 查看；本机用 `py` 调用）。
2. 在本文件夹打开终端，安装依赖：

```powershell
py -m pip install -r requirements.txt
```

3. 启动程序：

```powershell
py app.py
```

> 想先安全地看分类效果而不动文件，可运行：`py app.py --preview`

## 二、桌面整理怎么用

1. 左侧点【桌面整理】。
2. 点【① 扫描预览】——只看分类结果，**不会移动任何文件**。
3. 满意后点【② 一键整理】——把图标移动到桌面的分类文件夹（浏览器/办公/游戏…），**分类文件夹会自动在桌面左侧排整齐**。
4. 若图标位置仍不满意，点【⇅ 排列图标】可再次按顺序排列分类文件夹。
5. 不满意点【↩ 一键撤销】——把文件移回原来的位置；也可点【📋 历史记录】选某次还原。

安全说明：

- 系统图标（此电脑、回收站、网络、控制面板）会自动跳过，不会被动。
- 每次整理都会在 `%APPDATA%\桌面清理助手\history\` 留一份记录，撤销就靠它。
- 重名不会覆盖，会自动加后缀，如 `名字(1)`。

## 三、软件卸载怎么用（全部在本工具内完成，无需别的软件）

1. 左侧点【软件卸载】，点【刷新列表】。
2. 软件已自动分三类，默认只显示【可能想卸载】（杂七杂八、占空间的多在这）；
   顶部可切换【常用软件】【系统组件】【全部】。
3. **勾选**要卸载的软件（可多选；也可【全选本页】）。
4. 点【卸载选中】即可批量卸载。

两个选项（在按钮下方）：

- 【彻底卸载】：卸载后再删除残留的安装文件夹和注册表项（默认开启）。
- 【静默卸载】：尽量不弹出每个软件各自的卸载窗口（默认开启）。

注意：

- 卸载不可逆，操作前会二次确认。
- 标记「系统组件·受保护」的是系统/运行库，默认放在【系统组件】组，不要卸。
- 本程序已设为**以管理员身份运行**，双击时会弹 UAC 授权框，点【是】即可
  （卸载、清残留、移动被占用文件都需要管理员权限）。
- 病毒木马类请交给杀毒软件（Windows Defender / 火绒），本工具不负责查杀。

## 四、打包成 exe（可选）

```powershell
py -m PyInstaller --noconfirm --onefile --windowed --uac-admin --name DesktopCleanerAssistant ^
  --add-data "data;data" app.py
```

打包后在 `dist\` 下生成 exe，双击即可运行（无需再装 Python）。
`--uac-admin` 让它以管理员身份运行，双击会弹 UAC 授权框。

> 提示：PyInstaller 打的 exe 有时会被杀毒软件误报，属常见现象，可加信任或保留源码自查。

## 五、出问题怎么办

| 现象 | 处理 |
| --- | --- |
| 启动报缺少 customtkinter | 重新执行第一步的 `pip install -r requirements.txt` |
| 软件列表是空的 | 确认在 UAC 弹窗点了【是】（管理员身份） |
| 卸载没成功 | 取消勾选【静默卸载】，让它弹出各自卸载窗口手动完成 |
| 整理时有项目被跳过 | 关掉对应软件后重试；弹窗会显示具体原因 |
| 整理后分类文件夹位置乱 | 点【⇅ 排列图标】重新排整齐 |
| 整理后想全恢复 | 进【桌面整理】点【一键撤销】或【历史记录】（会连空文件夹一起清掉） |

## 六、使用发布包（exe）

1. 进入 `release\桌面清理小助手\` 文件夹。
2. 双击 **`桌面清理小助手.exe`**，UAC 点【是】即可使用。
3. 无需安装 Python，适用于 Windows 10/11，单机本地运行。
