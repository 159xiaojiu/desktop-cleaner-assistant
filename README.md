# 桌面清理小助手 (Desktop Cleaner Assistant)

面向 Windows 小白的桌面工具：**自动整理桌面图标** + **强力卸载顽固软件**。从真实使用痛点出发，用最小 MVP 解决「桌面乱」和「软件卸不掉」两件事。

> 完整产品规格见 [SPEC.md](SPEC.md) · 使用说明见 [使用说明.md](使用说明.md)

## 为什么做这个项目

| 痛点 | 解决方案 |
| --- | --- |
| 桌面几十上百个图标，找文件靠肉眼扫 | 按规则自动分类到「浏览器 / 办公 / 游戏…」文件夹 |
| 整理完不满意、怕删错 | 整理前预览 + 一键撤销 + 历史记录 |
| 控制面板卸不掉的流氓软件 | 集成 Bulk Crap Uninstaller，支持强力卸载与残留清理 |

## 产品亮点（作品集视角）

- **MVP 边界清晰**：只做整理 + 卸载，不做 360 式全家桶，控制范围避免无限膨胀
- **竞品调研驱动**：对标 coodesker、DeskFrame、BCU 等开源项目，明确「抄什么、不抄什么」
- **安全优先**：系统图标自动跳过、重名不覆盖、卸载二次确认、受保护组件标记
- **小白友好**：全程点鼠标，最终打包为双击即用的 exe

## 功能概览

**桌面整理**
- 扫描桌面图标 → 预览分类结果 → 一键整理 → 一键撤销

**软件卸载**
- 列出已安装软件，按「常用 / 可能想卸载 / 系统组件」分组
- 支持批量卸载、静默卸载、彻底清残留

## 快速开始（源码）

```powershell
py -m pip install -r requirements.txt
py app.py
```

预览模式（不移动文件）：`py app.py --preview`

## 技术栈

- Python 3.10+ · CustomTkinter
- PyInstaller 打包 · Windows UAC 管理员权限
- 卸载内核：Bulk Crap Uninstaller（开源集成）
